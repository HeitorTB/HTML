Dowload do pdf

  1. Colocar entre as tags a para transformar num link
  2. Dar ctrl + espaço e selecionar o arquivo no href
  3. Escrever dowload='nome do arquivo'
  4. Colocar o tipo do arquivo encontrado em: iana.org

Colocar áudio no site

  1. Baixar o audio na pasta do vs
  2. Colocar as tags audio
  3. Colocar depois da primeira tag audio o preload: metadata e o autoplay controls
  4. Colocar fora das tags source src + arquivo + type (audio/type)
    1. mp3
    2. ogg
    3. wav
