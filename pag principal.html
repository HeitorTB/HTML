Tabelas

A tag principal de abetura de tudo de dentro da tabela é table

O tr representa os elementos (representados por td ) dentro da linha horizontal

Com text align center e vertical align middle, consegue centralizar o conteudo da tabela

---------------------

No css colocar a borda (border) e a função: border colapse: colapse, para unir as linhas

As tabelas são divididas em: thead, tbody e tfoot (conclusão)

TR: Cada linha horizontal de assunto, TD: Os elementos dentro do tr, TH: elementos títulos

---------------------

Caption: Título da tabela

th scope row: elementos referidos estão na mesma linha horizontal (so utilizar em th)

th scope col: elementos referidos estão na mesma coluna vertical (so utilizar em th)

Tem como fazer um efeito zebrado na tabela: tbody > tr:nth-child(2n){determinado background color

2n: de duas em 2 linhas

---------------------

colspan: mesclar colunas

rowspan: mesclar linhas

Exemplo: td rowspan='3'

Editar colunas específicas: dentro de table criar um colgroup com varias 'col' dentro, cada uma representando uma coluna da tabela 

                colgroup
                    col class="coluna 1">
                    col class="coluna 2">
                    col class="coluna 3">
                /colgroup

Responsividade: Colocar toda a tabela dentro de um div e no css pôr dentro da div o overflow-x: auto

IFRAME

Tag: iframe

Dentro do src colocar o link da página para que apareça o frame do site

Coloque o link do lado de fora do src também, para que caso nao abra o frame, mostre o link

        iframe src="link da pag" frameborder="0"> link da pag <\iframe>

Frameborder='1' (com borda)

Frameborder='0' (sem borda)

-----------------------------

É possível colocar páginas internas do seu site no próprio iframe

Abrir um iframe específico na tela ao clicar no link:

  1. Cria um menu com diferentes páginas internas(link)
  2. Cria um iframe e define um --name--
  3. Coloca o name do iframe no target dos links do menu

Problema: O iframe enquanto não é escolhido fica em branco

Resolução: Colocar no lugar do src do iframe um srcdoc, e escrever algo dentro com html.

Opções de uso:

  1. Vídeos do youtube
  2. Google Maps (compartilhar--opção html)
  3. Mapas do Waze
  4. Google Docs/apresentações (arquivo--publicar na web)

Formulários

Tag: Form

fieldset > legend > form > label > input

Com a tag input cria-se as caixinhas de escrita

Com a tag input type:submit, cria-se o botão de enviar

Estrutura de um input: input type='', name='', id=''

Após criar a tag form, colocar dentro o action:banco de dados

Action: A tag action serve para anexar os dados após clicar no submit, a uma linguagem de banco de dados

-----------------------------

Sempre quando for uma caixa de texto, anexar o conteúdo numa tag label

Dentro do label terá um for, nele será colocado o id do input

Exemplo:

            form
            p> label for='inome'> Nome: <\label 
            input type='text' name='nome' id='inome'<\p
            /form

Os métodos são colocados dentro da tag form (method)

Existem dois métodos de formulários:

  1. Get: coloca o que foi inserido na url. Caso nao tenha sido escolhido o método, esse é o padrão
  2. Post: Não aparece na url, mas dá pra acessar com o inspecionar

--- Formulários get são compartilhaveis, formulários post não.

--- Usar o post com dados de grande porte, coisas sigilosas e envio de arquivos como fotos

-----------------------------

Colocar junto com o input o required, que serve para dar uma alerta para preencher o campo

Definir minimo: usar depois do required o min-length=''

Definir máximo: usar depois do required o max-length=''

Tag Placehoder: Serve para colocar uma indicação, colocado pós input

Tag Fieldset: Cria uma borda em volta do formulario (tag colocada fora do form)

Tag Legend: Coloca um espaço entre a borda do fildset e permite titular (tag colocada entre o form e o fildset)

-----------------------------

Ao invés de criar um input, tem a opção de criar select

As options são as opções que o select terá

form > label > select > options

Para criar categorias dentro de options, fazer uma tag: optgroup contendo todas as options da categoria, e criar uma label dentro da tag nomeando o tópico

Textarea: Tag que serve para o usuário deixar uma mensagem de maior porte
